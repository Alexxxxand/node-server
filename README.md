# Node Server

A simple NodeJS server for mini sites


Installing:

Installing NodeJS:


```
Debian:
sudo apt install nodejs
Arch:
sudo pacman -S nodejs
```



Cloning the repo:
```
git clone https://github.com/Alexxxxand/node-server
cd node-server
```

Installing the libraries:
```
npm i ejs
npm i express
```

Running:
```
node server
```
