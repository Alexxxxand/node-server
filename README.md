# Node Server

  A simple NodeJS server for mini sites



Cloning the repo:
```
git clone https://github.com/Alexxxxand/node-server
cd node-server
```

Installing the libraries:
```
npm i ejs
npm i express
npm i prompt-sync
```

Running:
```
node server
```
